# _ai-generated 디렉터리 안내

이 디렉터리는 AI Agent가 작업 중 생성하거나 재사용하는 보조 산출물을 둔다.

이 디렉터리 바로 아래의 `_scripts/`, `_snippets/`, `_exports/`, `_reports/` 등 밑줄로 시작하는 디렉터리는 저장소 전체에서 재사용할 수 있는 공통 산출물 영역이다.
`{companyName}/__{companyName}Common/` 하위에는 해당 회사 공통 영역 작업 중 생성한 보조 산출물을 둔다.
`{companyName}/_{domainName}/` 하위에는 해당 업무 도메인 작업 중 생성한 보조 산출물을 둔다.

민감정보, 운영 설정값, keystore, DB 파일은 명시적 요청 없이 이 디렉터리에 저장하지 않는다.
