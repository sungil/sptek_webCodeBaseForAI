# SQL Snippets

AI가 구현, 분석, 데이터 확인 과정에서 반복적으로 사용하는 읽기 전용 SQL을 보관한다.

- `common`: 테이블 목록, 컬럼 목록, 건수 확인처럼 DB 구조와 상태를 확인하는 범용 SQL
- `h2`: H2 전용 SQL
- `mysql`: MySQL/MariaDB 전용 SQL
- `{domain}`: 특정 업무 도메인에서 반복 사용하는 SQL
- `{feature}`: 특정 화면, API, 배치, 이벤트 분석에 필요한 SQL

SQL 파일에는 실제 비밀번호, 토큰, 개인 식별값, 운영 데이터 샘플을 저장하지 않는다.
