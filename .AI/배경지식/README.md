# 배경지식 디렉터리 안내

이 디렉터리는 AI Agent가 작업 대상을 이해하고 판단할 때 참고하는 배경지식 문서를 둔다.

이 디렉터리 바로 아래의 Markdown 파일은 저장소 전체에 공통으로 적용되는 배경지식이다.
`{companyName}/__{companyName}Common/` 하위 문서는 해당 회사 공통 영역을 이해할 때만 추가 참고한다.
`{companyName}/_{domainName}/` 하위 문서는 해당 업무 도메인을 이해할 때만 추가 참고한다.

배경지식은 모든 작업에 일괄 적용하지 않고, 현재 작업 범위와 관련성이 있을 때 선별적으로 읽는다.
