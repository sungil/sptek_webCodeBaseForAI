# 작업 수행 절차 디렉터리 안내

이 디렉터리는 AI Agent가 특정 작업을 수행할 때 따르는 절차 문서를 둔다.

이 디렉터리 바로 아래의 Markdown 파일은 모든 작업영역에서 공통으로 사용할 수 있는 절차다.
`{companyName}/__{companyName}Common/` 하위 문서는 해당 회사 공통 영역 작업 절차로만 사용한다.
`{companyName}/_{domainName}/` 하위 문서는 해당 업무 도메인 작업 절차로만 사용한다.

영역별 절차는 공통 절차를 대체하지 않고, 해당 영역에서 필요한 추가 단계나 검증 기준을 보완한다.
