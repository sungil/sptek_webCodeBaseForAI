# 예시프로젝트 참고기준

이 문서는 `_webFrameworkExample`을 실제 업무 개발에서 어떻게 참고할지 판단할 때 본다.

## 참고할 내용

`_webFrameworkExample`은 Framework코드영역 사용 방식을 보여주는 예시 업무 프로젝트다.
컨트롤러, 서비스, DTO, 보안 설정, 화면, profile 설정이 프레임워크 기능과 어떻게 연결되는지 확인할 때 참고한다.

프레임워크 동작을 이해할 때는 예시 코드만 보지 않고 `__webFramework`의 실제 처리 코드와 함께 본다.

## 그대로 옮기지 않을 내용

- 예시 URL
- 테스트용 권한 완화
- 샘플 데이터
- 임시 주석
- placeholder 성격의 패키지명, 화면명, 설정값

실제 업무 코드로 옮길 때는 업무 패키지, 업무 권한 정책, 실제 profile 설정 기준에 맞게 다시 정리한다.
