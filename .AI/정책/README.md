# 정책 디렉터리 안내

이 디렉터리는 AI Agent가 작업 중 반드시 지켜야 하는 정책 문서를 둔다.

이 디렉터리 바로 아래의 Markdown 파일은 모든 작업영역에 공통으로 적용한다.
`{companyName}/__{companyName}Common/` 하위 문서는 해당 회사 공통 영역 작업에서만 추가 적용한다.
`{companyName}/_{domainName}/` 하위 문서는 해당 업무 도메인 작업에서만 추가 적용한다.

영역별 정책은 공통 정책을 대체하지 않고 추가한다. 충돌이 불명확하면 작업 전에 사용자에게 확인한다.
