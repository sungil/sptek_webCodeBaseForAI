# SPT 코드 분석

SPT Framework Web Core 저장소의 소스 코드를 분석할 때 이 스킬을 사용한다. 이 파일은 Codex가 skill을 발견하고 진입하기 위한 최소 지침만 담는다.

## 사용 절차

1. 상세 분석 절차는 `.AI/workflows/skills/spt-code-analysis.md`를 따른다.
2. 검증이나 완료 보고가 필요한 경우 `.AI/workflows/common/verification.md`, `.AI/workflows/common/reporting.md`를 함께 따른다.
