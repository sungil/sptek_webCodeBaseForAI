# SPT Local H2 Query

SPT Framework Web Core 저장소의 `local` 프로파일 H2 file DB를 조회할 때 이 스킬을 사용한다.

## 사용 절차

1. 루트 `AI.md`를 먼저 따른다.
2. 상세 조회 절차는 `.AI/workflows/skills/spt-local-h2-query.md`를 따른다.
3. 빠른 조회 스크립트는 `.AI/workflows/scripts/query-local-h2.ps1`를 우선 사용한다.
4. 검증이나 완료 보고가 필요한 경우 `.AI/workflows/common/verification.md`, `.AI/workflows/common/reporting.md`를 함께 따른다.
