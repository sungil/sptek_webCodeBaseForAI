# 코드 분석

Framework코드영역과 그 관례를 바탕으로 작성되는 실제 업무 코드를 분석할 때 이 스킬을 사용한다. 이 파일은 Codex가 skill을 발견하고 진입하기 위한 얇은 adapter다.

## 사용 절차

- 코드 배치 경계가 관련되면 `.AI/배경지식/프로젝트-코드영역-구분.md`를 확인한다.
- `@Enable_*` 어노테이션이나 프레임워크 확장 지점이 관련되면 `.AI/배경지식/프레임워크-확장방식-확인기준.md`를 확인한다.
- 리소스나 profile 설정이 관련되면 `.AI/배경지식/리소스와-프로파일-구성.md`를 확인한다.
- `_webFrameworkExample`을 참고해야 하면 `.AI/배경지식/예시프로젝트-참고기준.md`를 확인한다.
