# Write Code Comments

코드 주석을 작성하거나 갱신할 때 이 스킬을 사용한다.

## 사용 절차

1. 루트 `AI.md`를 먼저 따른다.
2. 상세 절차는 `.AI/procedures/tasks/write-code-comments.md`를 따른다.
3. 커밋 전 변경 파일의 주석 점검은 `pre-commit-code-comments`가 이 스킬의 절차를 재사용한다.
