# 커밋 전 점검

커밋 전에 변경 범위, 주석 보강 필요 여부, 검증 상태를 함께 확인할 때 이 스킬을 사용한다.

## 사용 절차

- `_AI/_작업수행절차/코드변경-검증-절차.md`를 따른다.
- 주석 보강이 필요하면 `_AI/_작업수행절차/코드주석-작성-절차.md`를 따른다.
- 커밋 메시지는 `_AI/_작업수행절차/커밋메시지-작성-절차.md`를 따른다.
- `Git-작업-정책.md`를 따른다.
